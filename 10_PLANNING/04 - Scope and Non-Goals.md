# 범위와 제외 범위(Scope and Non-Goals)

## 1. 요약

이 문서는 끌리니 초기 기획에서 다룰 범위와 의도적으로 제외하거나 후순위로 둘 범위를 정리한다.

## 2. 현재 기획

- 기획서의 중심 범위는 무인 스터디카페/공간대여 시설의 이용 후 정리 자동화다.
- XLeRobot 기반 모바일 매니퓰레이터가 쓰레기 수거, 공간 정리, 자율주행, 기본 점검을 수행하는 방향이다.
- 관제 대시보드, 책상 닦기, 소등, 문단속은 기획서에 포함되어 있으나 MVP 수준과 구현 방식은 검토 필요다.

## 3. 왜 중요한가

범위가 넓어지면 로봇 하드웨어, 인식, 조작, 주행, 안전, 관제까지 모두 동시에 어려워진다. 초기 범위를 명확히 해야 데모 시나리오, Sprint 계획, 평가 기준을 현실적으로 정할 수 있다.

## 4. 핵심 내용

### 4.1 In Scope

| 항목          | 설명                                             | 상태      |
| ----------- | ---------------------------------------------- | ------- |
| 쓰레기 수거      | 책상·바닥의 쓰레기 후보를 탐지하고 지정 수거함으로 이동                | 구현 대상   |
| 공간 정리/정돈    | 의자, 테이블 위 소형 집기, 흐트러진 물체를 복구                   | MVP미포함  |
| 자율주행        | SLAM과 ROS 2 Navigation/Nav2 기반으로 작업 구역 이동 및 복귀 | 구현 대상   |
| 기본 점검       | 분실물 후보 확인, 소등, 문단속 등                           | MVP 미포함 |
| 책상 닦기       | 이용 후 책상 표면 정리 작업                               | MVP 미포함 |
| 온디바이스 추론 검토 | Jetson AGX Orin에서 객체 탐지, 위치 추정, 장애물 회피 수행      | 구현 대상   |
| 관제 대시보드     | 유사 서비스 비교표에 포함된 기능                             | MVP 미포함 |

### 4.2 Non-Goals

아래 항목은 초기 문서화를 위한 가정 초안이며, 팀 검토 후 확정해야 한다.

| 항목               | 제외 이유                                         | 상태      |
| ---------------- | --------------------------------------------- | ------- |
| 모든 무인점포 유형 완전 대응 | 업종별 공간 구조와 작업 대상이 다르므로 초기 범위가 과도해질 수 있음       | MVP 미포함 |
| 완전 자율 범용 로봇      | 기획은 무인 공간 이용 후 정리·점검에 초점을 둠                   | MVP 미포함 |
| 상용 수준 대규모 관제     | 관제 대시보드는 언급되어 있으나 대규모 운영 요구사항은 기획서에 상세하지 않음   | MVP 미포함 |
| 모든 자연어 명령 처리     | Rule-based VLA 중심이며 범용 자연어 명령 처리는 상세화되어 있지 않음 | MVP 미포함 |
| 법적/정책적 대응 완결     | 기획서에는 비용, 법률, 정책 이슈 placeholder가 있음           | MVP 미포함 |

### 4.3 범위 판단 기준 초안

- 이용 후 정리 시나리오와 직접 연결되는가?
- XLeRobot 기반 로봇으로 데모 가능한가?
- 평가 기준을 정의할 수 있는가?
- 안전 리스크를 관리 가능한 수준으로 줄일 수 있는가?
