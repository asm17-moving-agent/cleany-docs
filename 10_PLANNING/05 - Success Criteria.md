# 성공 기준(Success Criteria)

## 1. 요약

기획서 기반 성공 기준은 공간 정리, 자율주행, 쓰레기 후보 처리, 데모 시나리오 완수 여부를 중심으로 초안화한다. 정량 기준은 아직 추가 정의 필요다.

## 2. 현재 기획 이해

- 결과물 형태는 `XLeRobot 기반 무인 점포 관리 로봇 개발`로 기획서에 기재되어 있다.
- 기대효과는 공간 품질 유지, 고객 경험 개선, 다점포 운영 효율 향상, 야간/비상주 시간대 관리 공백 감소로 요약된다.
- 평가 지표의 수치 목표, 테스트 환경, 반복 횟수는 기획서에 상세하지 않다.

## 3. 왜 중요한가

성공 기준은 기능 우선순위, Sprint 목표, 데모 시나리오, 기술 검증의 기준이 된다. 정량 기준이 없으면 구현 완료 여부와 개선 방향을 판단하기 어렵다.

## 4. 핵심 내용

### 4.1 성공 기준 초안

| 영역      | 성공 기준 초안                                     | 정량 기준                     |
| ------- | -------------------------------------------- | ------------------------- |
| 공간 정리   | 이용 후 공간을 다음 이용자가 사용할 수 있는 상태로 복구한다.          | 추가 정의 필요                  |
| 쓰레기 처리  | 책상·바닥의 쓰레기 후보를 탐지하고 지정 수거함으로 이동한다.           | 추가 정의 필요                  |
| 분실물 후보  | 분실물 가능성이 있는 물체를 쓰레기와 구분해 보관 또는 알림 대상으로 처리한다. | 추가 정의 필요                  |
| 자율주행    | 작업 대상 구역까지 이동하고 작업 완료 후 대기 위치로 복귀한다.         | 추가 정의 필요                  |
| 정리/정돈   | 의자, 테이블 위 소형 집기, 흐트러진 물체를 정렬 또는 이동한다.        | 추가 정의 필요                  |
| 책상 닦기   | 책상 닦기 기능을 시연하거나 범위를 명확히 제외한다.                | 추가 정의 필요                  |
| 소등/문단속  | 상태 확인 또는 물리 조작 방식으로 점검한다.                    | 추가 정의 필요                  |
| 데모 시나리오 | 이용 종료 후 정리 자동화 흐름을 끝까지 보여준다.                 | 추가 정의 필요                  |
| 관제      | 작업 상태 또는 결과를 대시보드에서 확인한다.                    | 추가 정의 필요, MVP 포함 여부 검토 필요 |

### 4.2 결과물 후보

- XLeRobot 기반 무인 점포 관리 로봇
- 데모 영상
- 결과보고서
- 발표자료
- 시뮬레이션 또는 통합 테스트 결과

### 4.3 기대효과 연결

- 일정한 공간 품질 제공
- 고객 이탈 및 이미지 손실 감소
- 반복 순찰 및 청소 소요 감소
- 향후 로봇 임대, 유지관리 구독, 현장 설치/캘리브레이션 패키지 확장 가능성
