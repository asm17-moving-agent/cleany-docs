# 성공 기준(Success Criteria)

## 1. 요약

1차 MVP 성공 기준은 사전에 정한 쓰레기 물체를 분실물과 구분하고 안정적으로 집는 데 둔다. 자율주행·지도 생성·대시보드는 시연을 보강하는 요소로 다루며 정량 기준은 추가 정의가 필요하다.

## 2. 현재 기획 이해

- 결과물 형태는 `XLeRobot 기반 무인 점포 관리 로봇 개발`로 기획서에 기재되어 있다.
- 기대효과는 공간 품질 유지, 고객 경험 개선, 다점포 운영 효율 향상, 야간/비상주 시간대 관리 공백 감소로 요약된다.
- 평가 지표의 수치 목표, 테스트 환경, 반복 횟수는 기획서에 상세하지 않다.

## 3. 왜 중요한가

성공 기준은 기능 우선순위, Sprint 목표, 데모 시나리오, 기술 검증의 기준이 된다. 정량 기준이 없으면 구현 완료 여부와 개선 방향을 판단하기 어렵다.

## 4. 핵심 내용

### 4.1 성공 기준 초안

| 영역        | 성공 기준 초안                              | 정량 기준                         |
| --------- | ------------------------------------- | ----------------------------- |
| 쓰레기 분류·집기 | 사전에 정한 쓰레기 물체를 인식·분류하고 지정 수거함으로 이동한다. | 물체 종류, 반복 횟수, 집기 성공률 추가 정의 필요 |
| 분실물 후보    | 분실물 후보를 쓰레기와 구분하고 집거나 이동하지 않는다.       | 오분류 시 처리와 기록 방식 추가 정의 필요      |
| 현장 시연     | 쓰레기 분류·집기를 현장에서 보여준다.                 | 현장 시연의 최소 성공 조건 추가 정의 필요      |
| 자율주행      | 대상 구역 이동과 복귀를 보강 시연한다.                | 현장 시연 또는 영상 보완 조건 추가 정의 필요    |
| 지도·대시보드   | 초기 세팅, 공간 표시, 호출·상태 확인 흐름을 설명한다.      | MVP 포함 수준과 영상 범위 추가 정의 필요     |

### 4.2 결과물 후보

- XLeRobot 기반 무인 점포 관리 로봇
- 데모 영상
- 결과보고서
- 발표자료
- 시뮬레이션 또는 통합 테스트 결과

### 4.3 기대효과 연결

- 일정한 공간 품질 제공
- 고객 이탈 및 이미지 손실 감소
- 반복 순찰 및 청소 소요 감소
- 향후 로봇 임대, 유지관리 구독, 현장 설치/캘리브레이션 패키지 확장 가능성
