# 미해결 질문(Questions)

## 1. 요약

이 문서는 기획서 원문에서 비어 있거나 불확실한 항목, 문서 간 충돌, 추가 확인이 필요한 전제를 모아두는 질문 목록이다.

## 2. 현재 기획 이해

초기 KB는 기획서 원문과 사용자 제공 핵심 정보를 기반으로 작성되었다. 다만 팀 역할, 멘토 역할, 추진 일정, 수행 방법, 안전 기준, 평가 지표 등은 구체화가 필요하다.

## 3. 왜 중요한가

질문을 별도 문서에 남기면 AI나 작성자가 불확실한 내용을 임의로 확정하지 않고, Jira issue 후보 또는 Decision 후보로 연결할 수 있다.

## 4. 핵심 내용

### 4.1 기획 범위 질문

| 질문 | 배경 | 우선순위 | 상태 |
|---|---|---|---|
| 무인 스터디카페를 1차 타깃으로 확정할 것인가? | 기획서에는 1차 타깃으로 설정했다고 적혀 있으나 팀 합의 상태 확인 필요 | 높음 | 검토 필요 |
| 공간대여 시설의 구체 유형은 무엇인가? | 공간 구조에 따라 정리 대상과 안전 기준이 달라짐 | 중간 | 추가 확인 필요 |
| MVP 기능은 어디까지인가? | 쓰레기 수거, 공간 정리, 자율주행, 책상 닦기, 소등, 문단속, 관제의 범위 조정 필요 | 높음 | 검토 필요 |
| 관제 대시보드는 MVP인가, 향후 확장인가? | 유사 서비스 비교표에는 포함되어 있으나 상세 요구사항 없음 | 중간 | 검토 필요 |
