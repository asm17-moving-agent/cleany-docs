# 260715 - 로봇 프레임 구조

## 1. 결정

XLeRobot 상부 모듈과 4륜 Mecanum base를 연결하는 프레임 구조는 아직 선택하지 않는다. 다음 두 후보를 비교한 뒤 사람 검토를 거쳐 별도로 확정한다.

- XLeRobot 기본 구성에서 사용하는 IKEA RÅSKOG
- 매니퓰레이터 도달 범위와 목표 작업 높이에 맞춘 알루미늄 프로파일 frame

이 문서는 Decision 후보를 정의한 `draft`이며, 두 안 중 어느 것도 현재 선택된 결정이 아니다.

## 2. 이유

- 프레임 높이와 상부 모듈 배치는 듀얼 매니퓰레이터의 실제 작업 범위에 직접 영향을 준다.
- 4륜 Mecanum custom base와 상부 모듈을 물리적으로 결합할 mounting 구조가 필요하다.
- RÅSKOG 사용 시 XLeRobot 기본 조립 구성을 활용할 수 있는지, 알루미늄 프로파일 사용 시 목표 작업 높이에 맞춰 geometry를 조정할 수 있는지를 비교해야 한다.
- frame 변경은 base, 두 매니퓰레이터와 깊이 카메라 사이의 transform, 충돌 영역과 calibration 범위를 바꾸므로 플랫폼 선택과 분리해 검토해야 한다.

## 3. 대안

| 대안              | 검토 포인트                                                                         |
| --------------- | ------------------------------------------------------------------------------ |
| IKEA RÅSKOG     | XLeRobot 기본 구성 재사용 가능성, 실제 치수, 4륜 Mecanum base 체결 방식과 목표 작업 높이 충족 여부를 확인해야 한다. |
| 알루미늄 프로파일 frame | 높이와 mounting geometry를 조정할 수 있지만 설계·제작, 강성 검증, URDF와 calibration 작업 범위가 늘어난다.  |

## 4. 가정

- 상부 모듈은 듀얼 매니퓰레이터와 깊이 카메라를 유지한다.
- 이동 베이스는 selected Decision인 4륜 Mecanum custom base를 사용한다.
- RÅSKOG가 XLeRobot 기본 frame이라는 내용은 2026-07-15 사람 검토 입력을 기준으로 하며, 정확한 제품 사양과 조립 호환성은 추가 확인이 필요하다.
- 각 후보에서 매니퓰레이터 도달 범위, 작업 높이와 base 결합 geometry를 아직 실측하지 않았다.

## 5. 리스크

- RÅSKOG의 고정된 치수와 구조가 목표 작업 높이, 매니퓰레이터 도달 범위 또는 custom base mounting을 만족하지 못할 수 있다.
- 알루미늄 프로파일 frame은 설계·가공·조립 및 강성 검증 작업을 추가하고 표준 XLeRobot assembly 재사용 범위를 줄일 수 있다.
- LeRobot community의 자료와 도구가 SO-101 중심으로 최적화되어 있어, 듀얼 매니퓰레이터와 custom frame 구성의 예제 및 검증 자료가 부족할 수 있다.
- custom geometry는 base와 두 매니퓰레이터 사이의 기구학 모델, 충돌 검토 및 calibration 복잡도를 높일 수 있다.

## 6. 재검토 조건

- 두 후보의 실제 치수와 4륜 Mecanum base mounting 가능 여부를 확인한다.
- 목표 작업물의 높이·거리와 듀얼 매니퓰레이터의 도달 범위를 같은 좌표 기준으로 측정한다.
- 각 후보의 payload, 중심 높이, 강성, 조달 비용과 제작 일정을 비교한다.
- simulation 또는 mock-up에서 base-to-arm transform, 양팔 충돌 영역과 깊이 카메라 시야를 검증한다.
